# 🌍 Analysis of Air Quality Index (AQI) in Delhi

## 📌 Project Overview

This project focuses on analyzing Delhi's Air Quality Index (AQI) data using data science techniques. The objective is to uncover pollution trends, seasonal variations, and key environmental insights that impact public health.

## 🎯 Objectives

* Analyze AQI trends over time
* Identify major pollutants (PM2.5, PM10, NO2, etc.)
* Understand seasonal pollution patterns
* Perform Exploratory Data Analysis (EDA)

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Google Colab

## 📊 Key Insights

* High pollution levels observed during winter months
* AQI frequently falls under “Poor” and “Very Poor” categories
